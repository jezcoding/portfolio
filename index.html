HTML & CSS

Aside from Java, I’ve also started learning the basics of HTML and CSS, which are essential for web development. HTML (Hypertext Markup Language) is the standard language used to create the structure of web pages, while CSS (Cascading Style Sheets) is used to style them and make them visually appealing. With HTML, I learned how to use different tags like h1 to h6 for headings, p for paragraphs, and a for links. I can also create lists, insert images, and build forms. With CSS, I explored how to apply colors, fonts, margins, and layouts to a page. I’ve practiced creating simple websites with headers, footers, navigation bars, and responsive designs. What I enjoy most about HTML and CSS is seeing the immediate result of my code on the browser. It’s satisfying to take a blank page and turn it into something that looks clean and professional. These languages have helped me understand how websites are built from the ground up, and I’m looking forward to learning JavaScript next to make my websites interactive.

Python

Another language I’ve been exploring is Python, which has quickly become one of my favorites. Python is known for its simple and readable syntax, which makes it great for beginners like me. Even though I’ve only learned the basics, I already appreciate how powerful and flexible Python is. I’ve learned how to write simple Python scripts that involve variables, loops, if-else conditions, and functions. I also practiced using lists and dictionaries, which helped me understand how to manage data. One project I completed was a basic quiz app that asked users questions and scored their answers. Python’s simplicity helped me focus more on logic rather than complex syntax. I also learned that Python is widely used in data science, machine learning, and automation—fields I’m very interested in exploring in the future. For now, I’m working on improving my skills through small exercises and challenges, and I hope to take on more advanced projects soon.

Java

As a first-year Information Technology student at Cavite State University, one of the first programming languages I was introduced to is Java. I’ve learned that Java is a powerful, object-oriented language commonly used in building desktop applications, Android apps, and enterprise-level systems. At first, I found Java a bit challenging because of its strict syntax and structure, but over time, I started to understand the logic behind it. I have basic knowledge of Java, such as how to create variables, use loops and conditionals, and define classes and methods. One of the first programs I wrote was a simple calculator that could add, subtract, multiply, and divide numbers. That project helped me practice how Java handles input and output, as well as basic operations. What I like about Java is that it teaches you how to write organized and reusable code. Learning object-oriented concepts like classes, objects, and inheritance gave me a better understanding of how large programs are built. I know I still have a lot to learn, but I’m excited to continue improving my skills in Java
